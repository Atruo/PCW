Práctica 1

En esta práctica 1 de PCW, tuvimos como objetivo diseñar desde 0 una página web estática, usando los estándares de HTML para su creación y CSS para darle un estilo. En esta primera práctica al ser estática, todo apartado en el cual debiera haber una interacción directa con el usuario no se han implementado enteramente; habiendo por ejemplo botones que no realizan acciones.Por lo demás, logramos implementar la integridad de las funciones adicionales que se nos pidieron de forma correcta, como restricciones en los campos de texto, enlaces en distintas partes de la web o la correcta disposición de los contenidos. Tuvimos además en cuenta el diseño orientado a dispositivos móviles, modificando en tiempo real la disposición de la web y los menús de la misma en función de la resolución en la que se estuviera visionando.

Práctica 2

En esta practica 2 de PCW, nuestra intención era la de, partiendo desde una web estática que ya hicimos anteriormente, dotar a esta página de mayor interacción con el usuario. Informaríamos al usuario de las acciones que se están llevando a cabo y variaría dependiendo del resultado. Por otro lado, también trabajaríamos esta vez con contenido dinámico aportado por un servidor RESTful, para tener a nuestra disposición una base de datos con la trabajar, tanto para el almacenamiento de información de los usuarios como del contenido que se subiría; recetas en nuestro caso. Para conseguir la interacción con el usuario, implementamos en el código de la web JavaScript, sumado al css y al html anterior, los cuales hubo que cambiar ligeramente para que todo saliera correctamente. Para el servidor, nos ayudamos del servidor gratuito XAMPP, que venía implementado directamente en la máquina virtual. De todos los requisitos que se nos pidió en la práctica el único que no cumplimos fue el de conseguir almacenar una foto cuando creábamos una nueva receta. Por lo demás, todo se quedó en orden.
